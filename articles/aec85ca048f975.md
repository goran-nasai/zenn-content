## 目的

[前回の記事]()で紹介した大規模リニューアルについて、リニューアル後の実装と実態について

## 結果

Core Web Vitals におけるすべての指標で改善

- LCP
- CLS
- FID

:::message
FID については INP に完全にリプレイスされる
:::

### 改善幅

旧環境は WordPress x AWS。

- 画像圧縮

- Firebase Hosting

- Component 分割
